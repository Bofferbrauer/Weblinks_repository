  • Bitwarden

    • Since we didn't use a password manager during the course, I'm linking you to Bitwarden, which I use personally. It's free, it's open-source, and pretty powerful. It also comes with an extension for Chrome and Opera and possibly other browsers too, so retrieving your password is very easy. Just make sure it saved the right username and password, as it sometimes trips over itself here.

Text Editor

To write code, you need a text editor. Any editor would do, but it's much easier with a powerful one. VS Code is considered the best one right now, and that's what we're using.

  • VS Code

    • VS Code comes with many extensions to make it prettier and more powerful. Some of the best extensions are Prettier to make your code, well, prettier, open in browser to open the page in either your main or any other browser, VS Code Icons to have a clear overview which kind of page and/or folder you're handling, Auto Close Tags to close tags automatically and makes you win a lot of time, Bracket Pair Colorizer 2 to always see which opening bracket is linked to which closing bracket, and Better Comments to make comments stand out when you need to point at something special within the code. Additionally, I use the Panda Theme to have every element marked with different colors on a dark background, but if that's not your thing, there are tons of other themes that could better fit your interests.

    If you are styling your pages with SASS or SCSS, I can also recommend you the following extensions: HTML SCSS support to use SCSS, and both Live Sass Compiler(to compile sass into css) and Sass (for indenting, highlighting and auto-completing sass code) if you want to use SASS.

Basic Planning

If you want to create a professional website, you need to plan and prepare everything about it first. So here are the tools you need to prepare your Websites before you can actually start to make the site:

  • Gloomaps

    • Gloomaps is a tool which helps you plan how the different pages of your website will be linked together by allowing you to create a visual sitemap of your site and pages.

  • Draw.io

    • Draw.io allows you create a map of dependencies between the elements you have on your page, specifically the scripts from JavaScript.
    Careful!
    There is another website with the name draw.io that looks identical, but costs for being used, and it ain't exactly cheap. So make sure you use the site in the link and not the expensive knock-off.

  • Trello

    • Trello is a great tool to create a detailed to-do list, having an overview of every step in the process. Especially useful in teams, but also for solo developers.

Colors

A site needs colors, or it would be very boring. But you can't just take any color you want, they need to fit well together. To realize this, we need some tools that allow us to mix-and-match our colors perfectly together.

If you are working for a client, chances are that they have their set of colors that you are allowed to use, so in those cases there is no need to use a tool like these ones since the colors are preset already.

  • Coolors

    • A page's colors should complement each other and fit well together. With coolors you can generate a palette of colors to use for your website. You can reload the colors until you find a set that fits your needs and wishes for your website.

  • Adobe Colors

    • Similar to coolors in that you can find and create a palette of colors that suits your website. But here you have more direct control over which colors you can use and can see all the dependencies between the colors better. It is however more complicated to use, so for beginners coolors is generally recommended.

Layout

Before the Website can be made, one needs to know what the website is supposed to look like in the end. Nowadays, there are normally at least 2 layouts that need to made, one mobile and one for desktop. Adding a seperate layout for tablets is also a possibility that's often done. So the different layouts of the pages need to be set first before the page can actually get designed.

  • wireframe.cc

    • Wireframe.cc is a website that allows you to make a basic sketch of your website within just a couple of seconds. It's not very precise, but it allows to make a sketch of the design while listening to the demands and wishes of the client.

  • Balsamiq

    • Balsamiq allows you to create a precise sketch on how the website is supposed to look like in the end. With it, you can make pretty precise layouts without having to care about text or colors.
    Sadly, it's license is only 30 days free, after which you have to pay at least either $9 a month or $90 a year. But for a professional this can be a very useful tool and thus quickly pay for itself.

Fonts, icons and symbols

Each website needs a distinct font or set of fonts to differentiate itself from the competition. And pretty much every website also needs at least some icons or symbols to make the site look nice and easily readable. So here you can find some sources for these elements.

  • Google Fonts

    • Google Fonts is a free service provided by Google to obtain a font from a large list of possibilities. These fonts can either be downloaded or simply linked to their respective url. The list of fonts is very large and you should find there whatever you need. As is the case with the colors, a client can also tell you which font they want to use and provide it themselves too if it's not a standard font but a propietary one.

  • Font Awesome

    • Font Awesome can give you thousands of icons in a .svg format. What's special about them is that, like the name of the website alludes, those icons are actually considered as fonts, not svg pictures. As such, they need to be treated like text instead of images.

  • flaticon

    • flaticon is a website where you can find any icon you might need. While some need a premium account, the majority does notand there are so many you can be pretty sure you'll find what you're looking for. That goes double with the fact that most icons have several variations that can be downloaded either as an svg or png. In fact, you can actually download entire collections of a theme to ensure that all your icons have the same style and there's no odd one out.

Stock Pictures and Picture editing

Unless you have the right pictures already provided by your client for your website, you will either need to hire a photographer or rely on stock photography and stock pictures to fill your pages. There are plenty of websites that offer stock pictures, so if you can't find what you're looking for on one site, try again on another one. Be aware that you will have to mention their creators in your website credits if you use stock images.

  • Lorem Picsum

    • If you don't have a final picture but need to get a placeholder, you can get stock pictures specifically for this very need at Lorem Picsum. You don't need to credit anybody for these pictures as most of the time, they will not be part of the final design anyway.

  • Pexels

    • Pexels doesn't just offer stock images, but also stock videos if you need more animated content. All the pictures and videos are free and you even technically don't need to credit the creators, but it's always better to do so. They have a large library of pictures, so chances are that you can find what you're looking for.

  • StockSnap.io

    • Stocksnap.io offers a broad selection of free stock pictures, which can be sorted by themes and usage, like background for instance. Like with Pexels, crediting is not absolutely necessary but since it falls under best practices, better credit them anyway. You wouldn't want to be left out from mentioning the work you did either, would you?

  • Unsplash

    • Similar to Pexels, it comes with a very large collection of pictures. It has a very well sorted catalogue and you can quickly find a picture that suits your needs here. Unsplash aslo comes with a dedicated wallpaper section for backgrounds of any kind, be that for desktop computers, smartphones, tablets, cellphones or even apps.

  • remove.bg

    • One problem that most designs have if they don't have a rectangular shape is that they tend to have a solid background that in most cases needs to be removed. The same can of course happen with pictures which may have an undesirable background or something that might not be legal or free to show on your website. remove.bg can solve this problem easily for you, just drag and drop or ctrl-v your picture into the tool and it will automatically remove it's background.

  • pixlr

    • Pixlr is a free yet powerful online tool to edit your photos and to resize and reformat them. It comes in two flavors; pixlr X for basic photo editing functions, and pixlr E for more advanced photo editing functions. Some functions are hidden behind a paywall, but for more basic functions it's more than powerful and versatile enough. It also comes with a function to remove the background of your pictures in on click. And if all that wasn't enough yet, it also comes with a library of over 50,000 stock pictures, many of which are free to use. So all in all a very versatile tool that can be used for multiple functions at once.

  • tinyPNG

    • Photos and pictures formatted in PNG tend to be very large and thus can be a hindrance for websites, as they would take very long to load and quickly deplete the data plans for people with have a limit there. tinyPNG compresses the files by reducing the color depth by removing unused colors and removes all the unnecessary metadata. This results in pictures that are much smaller in weight, often down to a quarter without affecting the format or transparency.

  • Adobe Photoshop

    • Adobe Photoshop is a very powerful and thus widely used tool to edit photos, to the extent that photo editing is now colloquially known as "photoshopping". It's so well known that it doesn't really need any further description here.

    Nowadays Photoshop is part of an entire family of photo editing programs from Adobe, which also include Photoshop Lightroom, Photoshop Express and Photoshop Camera. Like all other Adobe products, it is part of the Adobe Creative Cloud, where you can use all Adobe products at once for a monthly fee. Many companies use the Creative Cloud, so chances are relatively high to use them on your future job, too.

Design and prototyping

So, now you have a basic layout, a color palette for your site, the right fonts and icons and some pictures to show on your website. Only now it's actually time to make a definitive design on how your website should look like. This has to include both desktop and mobile these days, specific tablet designs are also increasingly used, but you can in theory make as many design variants depending on the size of the screen as you want. With so many possible different variations, one needs powerful tools to handle all the different design versions at once.

  • Adobe XD

    • Adobe has a lot of products that are very helpful for any kind of design. To create Websites, Adobe XD is their most powerful and versatile tool. Not only can you create website designs for different screen sizes and breakpoints, you can also link the buttons you create there with other pages to simulate a browsing experience on the website. You can also create design elements like logos in Adobe XD to be used in the final website, but there are better tools within the Adobe suit to fulfill that role.

  • Figma

    • Figma is a web-based design tool that allows for entire teams to collaborate on the creation of a single design. It's not as resonsive as Adobe XD, but it's broader vision extends past webdesign and it's ability to create a design live with other people certainly more than makes up for that. This also means that you can not only create entire webpages, but of course also every visual element of the pages within Figma and use them later in your website.

  • Canva

    • While Canva definitely has better uses in webdesign than to prototype a website, you can create mobile and desktop website prototypes in Canva. But where Canva really shines in when it comes to create every single other design element, like logos, banners, emails and social media posts. The site comes with templates for each and every of these possibilities, complete with the exact measurements for the chosen website when it comes to social media inclusion. It's very easy to use, versatile and very powerful when you need design elements for and around your website.
